Das aktuelle Wetter, sowie die Wettervorhersage für die nächsten Tage. Dazu werden Daten über eine API abgefragt, zu JSON umgewandelt und grafisch aufbereitet. Zusätzlich gibt es eine automatische Standortabfrage oder Autovervollständigung in der Suche. Ist es in der gesuchten Stadt bereits Nacht ändert sich das Farbschema der Website automatisch.

Ein Echtzeit Chat auf der Basis von Google Firebase. Vor der ersten Benutzung müssen sich User per Firebase-Authentifizierung registrieren. Ohne registrierten Account kann der Chat weder gelesen, noch Räume gewählt oder darin geschrieben werden. Im Optionsmenü können User nach erfolgreicher Verifizierung ihr Passwort ändern oder den Account löschen.

Eine Sammlung umgesetzt mit React welche Daten grafisch in Karten darstellt und die entsprechenden Namen dieser in einer Suche auffindbar macht. Dazu wird nach jedem Tastendruck die Liste der verfügbaren Namen gegen die User Eingabe abgeglichen und die Karten neu gerendert. Zudem wurde durch einen State Hook ein Soundeffekt beim klicken der Karte realisiert.
